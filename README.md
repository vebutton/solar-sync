# solar-sync

solar-sync automates EV charger control against rooftop-solar **excess**
generation. It replaces the manual "watch the solar app, toggle the charger app"
loop with automation that supports configurable thresholds and a "mostly free
solar" charging mode that today's vendor apps don't expose. Built for a single
end user on an Emporia Level 2 EVSE + Base Power Company current tap; in the
viability-research phase.
